# Conversation Strategies

Landing page em Next.js/Tailwind para o produto Conversation Strategies.

## Estrutura

- `app/page.tsx`: pagina principal.
- `app/globals.css`: estilos globais e fundos.
- `public/images`: pasta reservada para imagens finais do site.

## Desenvolvimento

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
